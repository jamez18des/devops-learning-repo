## What is Linux?

Linux is an open-source operating system.

## Why do people use Linux?

- Security
- Stability
- Flexibility
- Open source
- Large community support

## Linux Distributions

Linux has different versions called distributions (distros).

Examples:
- Ubuntu (currently using)
- Debian
- Fedora
- Arch Linux
- CentOS

## Virtual Machines

When creating a virtual machine there are two common options:

### Virtualisation

- Faster
- Uses your computer's CPU directly
- Best when your hardware supports it

### Emulation

- Slower
- Simulates different hardware using software
- Useful for running different CPU architectures
